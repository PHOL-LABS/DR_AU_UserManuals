\section{Справочник команд \DRAudi{}}
Прошивка \DRAudi{} использует набор сервисных команд для чтения/записи параметров, диагностики и калибровки.
Доступность отдельных команд зависит от версии прошивки и аппаратной ревизии.

\begin{itemize}
    \item Команды конфигурации датчиков (скорость, температура, топливо, тахометр).
    \item Команды настройки индикации (яркость, режимы, предупреждения).
    \item Команды сервиса и диагностики (чтение состояния, сброс, проверка каналов).
\end{itemize}

\section{Примечания по совместимости}
\begin{itemize}
    \item Команды цветовой настройки активны только в ревизиях с поддержкой RGB.
    \item Для Bluetooth и Wi-Fi интерфейсов логика параметров одинакова, отличается только транспортный канал.
    \item Перед записью параметров рекомендуется сохранить рабочую конфигурацию.
\end{itemize}
