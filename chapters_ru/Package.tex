\chapter{Комплект поставки}\label{ch:package}

Каждый комплект \DRAudi{} включает:
\begin{itemize}
    \item собранный модуль приборной панели;
    \item программатор USBasp с внутренним кабелем;
    \item кабель miniUSB;
    \item датчик скорости (если предусмотрен вариантом);
    \item руководство пользователя и материалы по настройке.
\end{itemize}
