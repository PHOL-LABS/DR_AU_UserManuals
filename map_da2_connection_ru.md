# Подключение датчика давления во впускном коллекторе (MAP) к входу DA2

## Общая информация

Для измерения параметры L/100 на DR-Audi используется коннектор DA2 на основной печатной плате. 
Прошивка должна быть обновлена для поддержки внешнего датчика. 

Какие датчики можно использовать:
- Датчик ДАД 45.3829 АвтоГАЗ
- Fast FT54257 / Renault 223657266R (с доработками ПО)

Входной каскад датчика давления на плате DR-Audi реализован в виде
дифференциального усилителя на операционном усилителе.

Усилитель имеет:

-   положительный вход
-   отрицательный вход

Разность между этими сигналами формирует измеряемый сигнал.

При отсутствии входного сигнала операционный усилитель формирует
смещение, равное половине напряжения питания:

Voffset ≈ 2.5 В

Это значение используется как базовый уровень измерения.

------------------------------------------------------------------------

# Разъём подключения

Напряжение необходимо подавать на разъём:

DA2

Используемый коннектор:

WF-4 (2510-4P)

В стандартной комплектации разъём на плате отсутствует, присутствует
только посадочное место.

------------------------------------------------------------------------

# Подключение датчика

Типичный MAP-датчик выдаёт сигнал:

0 ... 5 В

Однако входной каскад имеет ограниченный динамический диапазон, поэтому
сигнал необходимо предварительно ослабить.

Для этого используется резистивный делитель.

------------------------------------------------------------------------

# Делитель напряжения

Рекомендуемая схема:

MAP sensor output \| \|---- R1 = 90k \| +----\> DA2 OUT+ \| \|---- R2 =
10k \| GND

Коэффициент деления:

K = R2 / (R1 + R2) K = 10k / 100k K = 0.1

В результате диапазон сигнала:

0 ... 5 В → 0 ... 0.5 В

------------------------------------------------------------------------

# Диапазон выходного сигнала усилителя

По результатам измерений:

  Vin      Vout
  -------- ---------
  153 мВ   2.573 В
  279 мВ   2.835 В
  365 мВ   3.000 В

При использовании делителя:

Vin = 0 ... 0.5 В

Ожидаемый диапазон на выходе усилителя:

Vout ≈ 2.25 ... 3.27 В

------------------------------------------------------------------------

# Диапазон значений АЦП

Если используется АЦП:

разрядность: 12 бит\
опорное напряжение: 3.3 В

перевод напряжения:

ADC = V / 3.3 \* 4095

Тогда ожидаемые значения:

Vmin ≈ 2.25 В → ADC ≈ 2790\
Vmax ≈ 3.27 В → ADC ≈ 4055

Рабочий диапазон измерений:

ADC ≈ 2800 ... 4050

------------------------------------------------------------------------

# Калибровка

Каждый MAP-датчик имеет собственную характеристику.

После оцифровки сигнала необходимо выполнить калибровку:

1.  измерить значения АЦП при нескольких известных давлениях
2.  построить зависимость:

Pressure = f(ADC)

3.  использовать линейную или полиномиальную аппроксимацию.

------------------------------------------------------------------------

# Итоговая схема

MAP sensor (0--5V) \| \| R1 90k +----///----+ \| +---- DA2 OUT+ \|
+----///----+ \| R2 10k GND

После делителя сигнал поступает на вход дифференциального усилителя,
затем оцифровывается АЦП микроконтроллера.
